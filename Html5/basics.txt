HTML - Hyper text markup language
Element- it is a name which is placed between < > is called as elements.
Tag- it is a type, in which an element is placed between < > angular brackets is called as tags.
ex: <h1></h1>

Structure of the HTML Document
---

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, Sanjay!</h1>
    <p>This is my very first webpage.</p>
</body>
</html>



---
Types of elements:

1)non-semantic : 
	ex: <head></head>,<body></body>
2)semantic: these provides additional and relevant information about the page.(used to search data by other systems).
	ex: <header>,<footer>,<nav>
---

div: divides a page into different divisions.
